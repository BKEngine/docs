runCmd¶

简介¶

调用一个bkscr指令。

原型¶
void runCmd(string name, dictionary param, function callback);
参数¶
参数 解释
name 调用的bks指令名
param 调用的参数列表
callback 函数真正执行完毕时触发的回调
说明¶

如调用wait*系列函数，则在wait的条件被触发、等待结束时callback回调才被触发。正常指令则立刻触发回调。

text指令则等待文字打印完成才触发回调。
